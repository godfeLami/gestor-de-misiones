# 🍥 Gestor de Tareas - Naruto

## 📌 Descripción
Aplicación web desarrollada en JavaScript que permite gestionar misiones ninja mediante un formulario interactivo.  
El sistema permite agregar, filtrar, completar y eliminar tareas, utilizando manipulación del DOM y almacenamiento local.

---

## 🚀 Funcionalidades

- ➕ Crear nuevas misiones
- 🔥 Asignar prioridad (Rango S, A, B)
- 📅 Seleccionar fecha de término (solo fechas válidas)
- ✔ Marcar misión como completada
- ❌ Eliminar misión
- 🔍 Filtrar por prioridad
- 📊 Contador dinámico de misiones
- 💾 Persistencia con localStorage

---

## 🛠 Tecnologías utilizadas

- HTML5
- CSS3 (Responsive + Animaciones)
- JavaScript (DOM + Eventos + Arreglos/Objetos)
- LocalStorage

---

## 🔐 Seguridad

- Sanitización de inputs para evitar XSS
- Uso de `textContent` en lugar de `innerHTML`
- Validaciones de entrada robustas

## 📱 Responsive

La aplicación es completamente responsive y funciona en:
